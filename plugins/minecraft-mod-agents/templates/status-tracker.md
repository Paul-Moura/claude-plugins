## Status Definitions

- `todo` - Ready to be picked up by a Coder
- `in-progress` - Actively being implemented
- `under-review` - Implementation complete, awaiting review
- `revision-needed` - Reviewer requested changes
- `design-escalation` - Design issue, returned to Architect
- `completed` - Approved and committed

## Workflow

```
todo → in-progress → under-review → completed
                          ↓
                   revision-needed → in-progress
                          ↓
                   design-escalation → (back to Architect)
```
